# Fibonacci app

## General info
This application allows to count Fibonacci number with three methods (recursive, iterative and the fast/matrix one) and compare theirs calculation times.

## Technologies
1. Java 8
2. Spring Boot 2.1.6
3. Thymeleaf 
4. Bootstrap 3.4
5. AOP
